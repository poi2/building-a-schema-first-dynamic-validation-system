# Articles / Blog Drafts

このディレクトリには、ブログやZennに投稿するための記事ドラフトを格納します。

## ディレクトリ構成

- プロジェクト内部のドキュメント → `docs/`
- 外部発信用の記事ドラフト → `articles/`

## 記事執筆ガイドライン

### ファイル名

`YYYY-MM-タイトル.md` の形式を推奨

例:
- `2026-02-schema-hot-reload-implementation.md`
- `2026-03-protovalidate-validation-strategy.md`

### 推奨される記事構成

1. **導入** - なぜこの機能が必要だったのか
2. **課題** - 解決したい問題
3. **解決策** - 採用したアプローチ
4. **実装詳細** - 技術的な実装のポイント
5. **動作確認** - デモや検証結果
6. **学び・まとめ** - 得られた知見

### 参照すべきプロジェクトドキュメント

- `docs/001-DD.xxx.md` - 設計ドキュメント（技術的な詳細）
- `docs/VERIFICATION-GUIDE.md` - 検証手順
- `DEVELOPMENT.md` - 開発環境のセットアップ
